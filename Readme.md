# Установка приложения

- на основе файла `config.sample.php` создать файл `config.php` и вписать свои настройки конфигурации
- накатить структуру БД, выполнив sql запросы из файла `schema.sql`
- накатить данные БД, выполнив sql запросы из файла `queries.sql`
